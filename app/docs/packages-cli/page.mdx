# @paljs/cli

## Introduction

A powerful command-line interface for generating full-stack applications with Prisma, GraphQL, and modern frontend frameworks. The PalJS CLI automates the creation of CRUD operations, admin interfaces, and GraphQL schemas.

## Usage

### `pal create <name>`

Create a new full-stack application with Prisma and GraphQL.

<PackageManagerTabs
  commands={{
    npm: "npx pal create my-awesome-app",
    yarn: "yarn pal create my-awesome-app",
    pnpm: "pnpm pal create my-awesome-app",
  }}
/>

**Options:**

- `--example <type>` - Choose project template (default: `full-stack-nextjs`)
- `--framework <framework>` - UI framework selection
- `--multi` - Enable multi-schema support
- `--git` - Initialize git repository
- `--manager <manager>` - Package manager (npm, yarn, pnpm)
- `--skip-install` - Skip dependency installation

**Available Examples:**

- `full-stack-nextjs` - Complete Next.js application with Prisma
- `apollo-nexus-schema` - Apollo Server with Nexus schema-first approach
- `apollo-sdl-first` - Apollo Server with SDL-first approach
- `graphql-modules` - GraphQL Modules architecture

**Available Frameworks:**

- Material UI
- Material UI + PrismaAdmin UI
- Tailwind CSS
- Tailwind CSS + PrismaAdmin UI
- Chakra UI
- Chakra UI + PrismaAdmin UI

**Example:**

<PackageManagerTabs
  commands={{
    npm: 'npx pal create my-app --example full-stack-nextjs --framework "Tailwind CSS + PrismaAdmin UI" --git',
    yarn: 'yarn pal create my-app --example full-stack-nextjs --framework "Tailwind CSS + PrismaAdmin UI" --git',
    pnpm: 'pnpm pal create my-app --example full-stack-nextjs --framework "Tailwind CSS + PrismaAdmin UI" --git',
  }}
/>

### `pal generate [models] [type]`

Generate CRUD operations, admin pages, and GraphQL queries/mutations.

<PackageManagerTabs
  commands={{
    npm: "npx pal generate",
    yarn: "yarn pal generate",
    pnpm: "pnpm pal generate",
  }}
/>

<PackageManagerTabs
  commands={{
    npm: "npx pal generate User,Post",
    yarn: "yarn pal generate User,Post",
    pnpm: "pnpm pal generate User,Post",
  }}
/>

## Command Reference

### Commands

| Command    | Description              | Example             |
| ---------- | ------------------------ | ------------------- |
| `create`   | Create new project       | `pal create my-app` |
| `generate` | Generate CRUD operations | `pal generate User` |
| `schema`   | Schema utilities         | `pal schema json`   |

### Options

| Option      | Description    | Default |
| ----------- | -------------- | ------- |
| `--help`    | Show help      | -       |
| `--version` | Show version   | -       |
| `--verbose` | Verbose output | `false` |

## Features

- **Project Scaffolding**: Generate full-stack applications with modern best practices
- **CRUD Generation**: Automatic creation of GraphQL operations
- **Multiple Frameworks**: Support for popular UI frameworks
- **Type Safety**: Full TypeScript support throughout
- **Flexible Templates**: Choose from multiple project templates
- **Package Manager Agnostic**: Works with npm, yarn, and pnpm
